# map

不会改变原数组,返回新的数组, 类似加工，加工每个项

let arry = [1, 2, 4, 8]
let Arry = arry.map(v => v * 10)
console.log(arry) // [1, 2, 4, 8]
console.log(Arry) // [10, 20, 40, 80]
let Arry2 = arry.map(v => v > 7) // 返回判断的结果
console.log(Arry2) // [false, false, false, true]

# filter

不会改变原数组,返回新的数组, 过滤器，类似漏斗，小沙子过滤出来，大沙子留住

let arry = [1, 2, 4, 8]
let Arry = arry.filter(v => v * 10)
console.log(arry) // [1, 2, 4, 8]
console.log(Arry) // [1, 2, 4, 8] 不会执行运算操作
let Arry2 = arry.filter(v => v > 7) // 返回过滤后符合的结果
console.log(Arry2) // [8]
let Arry3 = arry.filter(v => v > 10)
console.log(Arry3) // [] 过滤后如果没有符合的项，返回空数组

# reduce

累加器，回调函数包含4个参数，第一个是上一次执行的结果，初次默认是数组第一个值，第二个参数是当前数组轮到被处理的值，第三个是该被处理值的下标，第四个是调用reduce的数组

let arry = [1, 2, 4, 8]
let res = arry.reduce((sum, v, index, arr) => {return sum + v})
console.log(arry) // [1, 2, 4, 8]
console.log(res) // 15
