# 组件按需引入

以ant-design-vue组件为例

index.js文件

import Vue from 'vue'

// 这里引入项目中用到的ant组件
import {
  Button,
  Message
  // 其他组件
} from 'ant-design-vue'

// 用一个数组保存引入的组件，目的是为了方便循环注册
const antdComponents = [
  Button
  // 其他组件
]

// 像message这些全局消息，直接挂在原型上，这样就不用每个用到message的页面都再次进行import组件
Vue.prototype.$message = Message

// 循环注册
antdComponents.forEach(ant => {
  Vue.component(ant.name, ant)
})

引入

main.js中import这个index.js

当然仅仅是这样还不够的,需要配合插件【babel-plugin-import】按需引入

安装好babel-plugin-import插件，在babel.config.js文件进行配置

 plugins: [
    // 配置 babel-plugin-import,按需引入资源
    [
      'import',
      { libraryName: 'ant-design-vue', libraryDirectory: 'es', style: true, javascriptEnabled: true }
    ]
  ]

此外还不够，antd用到了less相关的样式定义，所以还要配置一下css-loader相关的东西

在vue.config.js文件中配置

module.exports = {
  devServer: {
    port: 7777 // 自定义启动端口
  },
  css: {
    loaderOptions: { // 向 CSS 相关的 loader 传递选项
      less: {
        javascriptEnabled: true // 引入antd-ui的时候要配置这个，不然报相关less-loader的错
      }
    }
  }
}
