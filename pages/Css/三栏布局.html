# 三栏布局

# 1. 弹性布局（flex布局）

注意：main要设置 flex: 1 自适应

<div class="container">
    <div class="left">左</div>
    <div class="main">中</div>
    <div class="right">右</div>
</div>

.container{
  display: flex;
}
.left{
  width: 200px;
  background: red;
}
.main{
  flex: 1;
  background: pink;
}
.right{
  width:200px;
  background: red;
}

# 2. 浮动布局（float + margin）

注意：main要放在最后

<div class="container">
    <div class="left">左</div>
    <div class="right">右</div>
    <div class="main">中</div> <!-- 注意中间内容要写在最后！！！-->
</div>

/*左边栏左浮动*/
.left{
  float: left;
  height: 100%;
  width: 200px;
  background: red;
}
/*中间栏自适应*/
.main{
  height: 100%;
  margin:0 200px;
  background: pink;
}
/*右边栏右浮动*/
.right{
  float: right;
  height: 100%;
  width: 200px;
  background: red;
}

# 3. 绝对布局（position）

注意： 左右分别绝对定位，中间可通过margin或者也是绝对定位

<div class="container">
    <div class="left">左</div>
    <div class="main">中</div>
    <div class="right">右</div>
</div>

/*左右进行绝对定位*/
.left, .right{
  position: absolute;
  height: 100%;
  top: 0;
  background: red;
}
.left{
  left: 0;
  width: 200px;
}
.right{
  right: 0;
  width: 200px;
}
/*中间用margin空出左右元素所占的空间*/
.main{
  height: 100%;
  margin: 0 200px;
  background: pink;
}
/*或者中间也进行绝对定位*/
.main{
  position: absolute;
  height: 100%;
  left: 200px;
  right: 200px;
  background: pink;
}

# 4. 表格布局（table）

注意，container要设置width

<div class="container">
    <div class="left">左</div>
    <div class="main">中</div>
    <div class="right">右</div>
</div>

.container{
    display: table;
    width: 100%; // 这里一定要设置width，否则不生效
}
.container > div{
    display: table-cell; // 三栏都要设置table-cell
}
.left{
  width: 200px;
  background: red;
}
.main{
  background: pink;
}
.right{
  width: 200px;
  background: red;
}

优点：兼容性很好（ie8 及以上） 父元素高度会被子元素撑开（不担心高度塌陷）

缺点: seo 不友好 当其中一个单元格高度超出的时候，其他的单元格也是会跟着一起变高的

# 5. 网格布局（grid）

注意： 设置列数的width值，三列就设置3个，auto为中间自适应

<div class="container">
    <div class="left">左</div>
    <div class="main">中</div>
    <div class="right">右</div>
</div>

.container{
  display: grid;
  width: 100%; // 默认是100%，按需设置
  grid-template-rows: 100px;  /*设置行高*/
  grid-template-columns: 200px auto 200px;  /*设置列数属性*/
}
.left{
  background: red;
}
.main{
  background: pink;
}
.right{
  background: red;
}
