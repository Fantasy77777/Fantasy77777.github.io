# 移动端适配

在index.html编辑viewport

<meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
1

width=device-width 设置视口宽度等于设备的宽度

user-scalable：用户是否可以手动缩放

initial-scale： 初始缩放比例, 1不缩放

maximum-scale：允许用户缩放到的最大比例

minimum-scale：允许用户缩放到的最小比例

安装px2rem-loader插件

  1. yarn install px2rem-loader --save-dev

  2. 在vue.config.js进行对应的配置

css: {
  // css预设器配置项
  loaderOptions: {
    postcss: {
      plugins: [
        require('postcss-px2rem')({
          remUnit: 100 // 将html字体设置为100px,这样0.1rem就是10px了，方便计算，如果设置为375px,计算就不是很方便
        })
      ]
    }
  }
}

  1. 在 main.js 设置 html 根字体大小
function initRem() {
  let cale = window.screen.availWidth > 750 ? 2 : window.screen.availWidth / 375
  window.document.documentElement.style.fontSize = `${100 * cale}px`
}

// 窗口变化就进行监听
window.addEventListener('resize', function() {
  initRem()
})
