# 针对不同的浏览器或浏览器不同版本写相应的CSS的过程，就是CSS hack

分类

css hack有种：条件hack、属性级hack、选择符hack

条件hack

/* 只在IE中有效 */
<!--[if IE]>
<style>
	.test{color:red;}
</style>
<![endif]-->

/* 大于IE6 */
<!--[if gt IE 6]>
<style>
.test{color:red;}
</style>
<![endif]-->

/* 大于等于IE6 */
<!--[if gte IE 6]>
<style>
.test{color:red;}
</style>
<![endif]-->

/* 小于IE6 */
<!--[if lt IE 7]>
<style>
.test{color:red;}
</style>
<![endif]-->

/* 小于等于IE6 */
<!--[if lte IE 7]>
<style>
.test{color:red;}
</style>
<![endif]-->

/* 非IE7 */
<!--[if ! IE 7]>
<style>
.test{color:red;}
</style>
<![endif]-->

属性hack（类内部hack）

.test{
	color:#090\9; /*For IE8*/
	*color:#f00; /*For IE7 and earlier*/
	_color:#ff0; /*For IE6 and earlier*/
}

* htm .test{color:#0f90;} /*For IE6 and earlier*/
* + html .test{color:#ff0;} /*For IE7*/
